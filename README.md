The curated T2DM metabolic model is provided in Systems Biology Markup Language (SBML) format. The model was exported using the RAVEN Toolbox \cite{Agren12013},\cite{ElenaLucyCarter2024} and includes nine cellular compartments: \textit{extracellular} (s), \textit{Peroxisome} (p), \textit{Mitochondria} (m), \textit{Cytosol} (c), \textit{Lysosome} (l), \textit{Endoplasmic reticulum} (r), \textit{Golgi apparatus} (g), \textit{Nucleus} (n), and \textit{Boundary} (x). The model contains 6,846 species classified as metabolites (M\_mxxxx) and enzymes (ENSG identifiers), and 5,590 reactions, including 1,732 reversible reactions and 3,858 irreversible reactions. Parameter fields are intentionally left undefined to allow users to incorporate context-specific kinetic or experimental data.
